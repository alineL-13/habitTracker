# Habit Tracker

A full-stack habit tracking web application built to deepen my understanding of backend architecture, authentication, API design, and state persistence — while also improving code organization using Vanilla JavaScript.

## Live Demo
App: https://habittracker-1-s6yw.onrender.com/index.html
- This project is hosted using free-tier services for learning purposes.
- The backend may take a few seconds to wake up after inactivity.

## Features
- Authentication
- User sign up and login
- Password hashing using bcrypt
- JWT authentication
- Protected routes
- Login persistence for 7 days
- User-specific data isolation
- Habit Management
- Create habits
- Edit habit name and color
- Delete habits
- Mark habits as completed

## Visualizations
### Weekly View
- Navigate between weeks
- Habit completion tracking
- Weekly progress bar
### Monthly View
- Navigate between months
- Monthly completion overview
### Manage View
- View habit statistics
- Edit habits
- Add new habits
- Remove habits
